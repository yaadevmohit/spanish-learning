# Spanish Learning App

A custom Spanish learning platform designed to provide a structured path through curated YouTube content and interactive exercises.

## Overview

This application helps users master Spanish by organizing educational YouTube videos into a logical learning progression. Each video is embedded directly into the platform and followed by a set of targeted exercises to reinforce the concepts covered.

## Key Features

- **Curated Learning Path:** A step-by-step journey through Spanish grammar, vocabulary, and culture.
- **Integrated Video Lessons:** Embedded YouTube videos for a seamless learning experience.
- **Interactive Exercises:** Practice what you've learned immediately after watching each video.
- **Progress Tracking:** (Planned) Keep track of your journey as you complete each lesson.
